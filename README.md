Данный скрипт (компонент `QuizController`) реализует **интерактивную систему тестирования (викторины)** для проектов на движке UNIGINE. Он объединяет графический интерфейс на базе `ObjectGui` и физические кнопки, что делает его одинаково удобным как для классических ПК-приложений, так и для VR-сред или игровых автоматов с реальными тактильными элементами.

**Основные возможности:**
- Полностью настраиваемые вопросы через редактор: текст вопроса, четыре варианта ответа, индекс правильного варианта.
- Отображение вопросов и ответов в GUI в виде сетки 2×2.
- Физические кнопки (ноды), срабатывающие при смещении по оси Z (порог настройки).
- Визуальная обратная связь: ✓ (зелёный) при правильном ответе, ✗ (красный) при ошибке.
- Автоматическая задержка перед следующим вопросом (настраивается).
- Подсчёт правильных ответов и сравнение с заданным проходным баллом.
- Методы для внешнего управления: `StartQuiz()`, `HideQuiz()`, `ShowQuiz()`, `IsQuizPassed()`.

**Применение:** Скрипт идеально подходит для обучающих симуляторов, VR-тренажёров, интерактивных стендов и аркадных автоматов, где требуется проверка знаний с использованием физического взаимодействия. Благодаря модульной архитектуре легко расширяется и интегрируется в существующие проекты.
